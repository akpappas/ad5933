Parameters
measurementsa vector of pairs. The pairs contain the frequency of a measurement and the value that was measured.
calibration_resistancethe value in Ohms of the calibration resistance. The vector MUST contain the measurements of the known calibrations resistance for the gain calculation to be meaningful.
Returns
A vector of pair, that contain the frequency and the gain factor for that frequency.
